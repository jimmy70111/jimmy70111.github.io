Technologies Used

AWS (Amazon Web Services)

Hosting: Utilize AWS services to host the application with high availability and reliability.

Spring Boot with Maven

Framework: Use Spring Boot to build a robust, production-ready application. It simplifies the development of Java-based microservices.

Dependency Management: Manage project dependencies and build processes with Maven. It aids in organizing the project and handling library versions.

Microservices Architecture

Modular Design: Decompose the application into distinct microservices to handle various functionalities:

  • User Management: Handle user registrations, logins, and profiles.
  • Item Management: Manage item listings, descriptions, and pricing.
  • Shipping Service: Track and manage shipping information.
  • Payment Service: Oversee transactions and payment processing.

Integration

APIs: Define RESTful APIs to enable communication between microservices and the front-end.

Database Design

MySQL: Manage data related to items, users, orders, and payments.

Back-End Services (Java)

Spring Boot Microservices: Create microservices for different application aspects:

  • Item Service: Perform CRUD operations for items.
  • Order Service: Manage customer orders and their statuses.
  • Shipping Service: Track and manage shipping details.
  • Payment Service: Process payments and manage payment information.
